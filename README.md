# LeoMp3Parser  
Identifies URL's to Audiofiles on Leo Dictionary (https://dict.leo.org/englisch-deutsch/)


## Development

Start by installing bundler:

    gem install bundler

And then install the dependencies:

    bundle install

Run the test suite:

    rake

or

    rspec
